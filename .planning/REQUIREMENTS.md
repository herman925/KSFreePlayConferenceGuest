# Requirements: KSFreePlayConferenceGuest

**Defined:** 2026-03-12
**Core Value:** Participants can quickly find schedule, activity details, and survey links during the conference

## v1 Requirements

### Navigation

- [x] **NAV-01**: Participant sees a clean homepage with clear cards/sections for all content areas
- [x] **NAV-02**: Site works well on mobile (participants will use phones during conference)
- [x] **NAV-03**: Navigation back to homepage is always accessible

### Programme

- [ ] **PROG-01**: Participant can view the full programme rundown / schedule with times
- [ ] **PROG-02**: Participant can download or view the Programme PowerPoint (link: https://qr.link/2lBfdx)

### Activity Details

- [ ] **ACT-01**: Participant can view Activity 1 instructions and details
- [ ] **ACT-02**: Participant can view Activity 2 instructions and details
- [ ] **ACT-03**: Participant can download Guideline document (link: https://qr.link/inG2Km)

### Surveys & Classroom

- [ ] **SURV-01**: Participant can access pre-test survey (https://eduhk.au1.qualtrics.com/jfe/form/SV_1UgFScb9EjCdZPw)
- [ ] **SURV-02**: Participant can access post-test survey (https://eduhk.au1.qualtrics.com/jfe/form/SV_6sNQjkbijFtMiX4)
- [ ] **SURV-03**: Participant can join Google Classroom (https://qr.codes/KcWzCB)
- [ ] **SURV-04**: Survey section is prominently placed and clearly labelled on the homepage

## v2 Requirements

### Enhancements

- **ENH-01**: QR code display for each survey link (for quick phone scanning)
- **ENH-02**: Countdown timer to next session

## Out of Scope

| Feature | Reason |
|---------|--------|
| Transport / logistics | Participants have already arrived |
| Budget page | Organizer-only |
| Reporting page | Organizer-only |
| Tasks / role selector | Participants are homogeneous audience |
| Participants list | Privacy concern, not needed by attendees |
| Server-side functionality | GitHub Pages is static only |
